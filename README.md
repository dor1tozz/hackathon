Проект разработан в рамках хакатона НИУ ВШЭ по предмету "Технологии программирования".
Программа автоматичиски сортирует электронные письма и классифицирует их по категориям, а также формирует отсчет о результатах работы.

Что может программа:
Чтение писем в формате .eml из папки inbox
Классифицирует письма по категориям
Обрабатывает также ошибочные файлы и отбрасывает их
Формирует JSON отсчет
Запуск осуществляется через Bash скрипт (run.sh)

На какие категории писем рапределяет:
-critical_incident
-support_request
-spam
-sent
-draft
-billing
-hr
-informational
-uncategorized

Как сделать скрипт исполняемым "- сhmod +x run.sh"
Запустить программу "./run.sh'
На выходе после запуска получаем:
1) Программа считывает письма из папки
2) Классифицирует каждое по категориям
3) Копирует письма в соответсвующие папки
4) Делает журнал обработки
5) Создает JSON - отсчет со статистикой
